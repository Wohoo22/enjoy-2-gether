# Enjoy2gether

A website for finding groups to share Netflix accounts.

# Functions for users:
* Login, register
* Create group
* Join group
* Posting and commenting 
* Charge the balance

# Functions for admin:
* Kick users
* Access all groups
* Update user's balance
* Post, comment in all groups
* Provide Netflix accounts

Use this command to run the project: 
```
mvn tomcat7:run
```
